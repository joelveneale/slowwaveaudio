Project Title 3

Interactive Sound Design | 2024

Project Overview

This project involved creating an adaptive audio system for an interactive installation piece. The challenge was to design sounds that would respond dynamically to visitor interactions while maintaining musical coherence.

We developed a generative system that could create infinite variations while staying true to the artistic vision of the installation.

▶

Interactive Design

Using motion sensors and proximity detection, we created an audio environment that evolves based on visitor movement and engagement. Each zone within the installation space triggered different sonic layers and textures.

The system was built to run continuously for the duration of the exhibition, with algorithms ensuring that no two visits would be exactly the same.
