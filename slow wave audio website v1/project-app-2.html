Generative Music Sequencer | Web App

About the App

A browser-based generative music sequencer that allows users to create evolving musical patterns through an intuitive visual interface. Perfect for ambient music creation, meditation, and creative exploration.

The application uses algorithmic composition techniques to generate infinite variations based on user-defined parameters and scales.
